How Does The Game Work?

This game was created as a final project for CSIS 1430. To view a detailed explanation on how the game works as well as how to do this yourself go back to the "game" page. Once there you can right click anywhere on the page and click the "Source Code" button. This will display the source code along with comments detailing most lines of code. Not only was this done for the viewer but I have a tendency to forget why I wrote lines the way I did and often find myself questioning why I didn't write it differently only to spend several minutes reading through the code to find out I needed to do it a specific way to account for something else I did. JavaScript is not a language I use often so I know there are ways I could have improved on along with features I could have added. I am by no means a designer so while the website may not be beautiful, it functions exactly how intended which is more-so my goal than trying to design something visually appealing.
